# Presentation Cases

This folder contains curated demo images chosen for stable robot-friendly presentation cases.

Recommended pipeline:

- `classical` perception
- `two_opt` planning

## Cases

- `01_vertical_line`: Vertical Line - Single clean vertical stroke.
- `02_horizontal_line`: Horizontal Line - Single clean horizontal stroke.
- `03_cross`: Cross - Two straight strokes crossing at the center.
- `04_filled_square`: Filled Square - A filled colored square whose contour should be extracted once.
- `05_filled_triangle`: Filled Triangle - A filled colored triangle for clean contour extraction.
- `06_filled_circle`: Filled Circle - A filled colored circle that should become one closed contour.
- `07_house_badge`: House Badge - A simple filled house-style badge made of colored regions.
- `08_filled_diamond`: Filled Diamond - A filled colored diamond for a single clean contour.
- `09_geometric_logo`: Geometric Logo - Triangle, square, and circle arranged as filled colored logo parts.
- `10_monogram_blocks`: Monogram Blocks - Three separated filled shapes similar to a simple logo mark.
- `11_filled_star`: Filled Star - A filled five-point star for a more presentation-friendly contour demo.
- `12_filled_pentagon`: Filled Pentagon - A filled pentagon that should become one clean polygon contour.
- `13_filled_hexagon`: Filled Hexagon - A filled hexagon that stays simple for the robot to draw.
- `14_arrow_badge`: Arrow Badge - A filled arrow-shaped badge with one bold contour.
- `15_shield_badge`: Shield Badge - A filled shield-style emblem for a single closed outline.
- `16_lightning_mark`: Lightning Mark - A filled lightning-bolt style symbol with one strong silhouette.
- `17_ribbon_badge`: Ribbon Badge - A filled award-ribbon silhouette with one strong outer contour.
- `18_crown_icon`: Crown - A filled crown silhouette that stays clean in both classical and ML extraction.
- `19_bookmark_icon`: Bookmark - A filled bookmark silhouette with a simple notched base.
- `20_leaf_icon`: Leaf - A clean filled leaf silhouette without inner detail.
